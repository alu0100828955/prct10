que permita a\-pro\-xi\-mar el número $\pi$ con una cierta precisión.
%
$\pi$ se puede calcular mediante integración:

$$\int_{0}^{1} \! \frac{4}{1+x^2}\, dx = 4(atan(1) -atan(0)) = \pi $$
